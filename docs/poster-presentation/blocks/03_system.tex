\begin{block}{System Architecture}
  A local \textbf{control plane} (FastAPI + Next.js + MLflow + Python runner) launches
  experiments and stores results. Every model is served with \textbf{Docker + vLLM}
  behind an \textbf{OpenAI-compatible HTTP} endpoint, and a \textbf{metrics proxy}
  records cost, energy, CO\textsubscript{2}, tokens, and latency on each call.
