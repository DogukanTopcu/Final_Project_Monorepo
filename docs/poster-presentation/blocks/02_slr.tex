  \vspace{4pt}
  \textbf{Research questions.}
  \begin{itemize}
    \item \textbf{RQ1 Performance:} hybrid / multi-agent vs.\ monolithic LLM.
    \item \textbf{RQ2 Efficiency:} effect on cost, latency, and energy.
    \item \textbf{RQ3 Orchestration:} routing, verification, coordination mechanisms.
    \item \textbf{RQ4 Domain suitability:} where SLM-centred designs win.
  \end{itemize}

  \textbf{Key gap motivating this thesis:} accuracy is widely reported, but
  \emph{cost and energy are not} --- so architectures are rarely compared on a
  common efficiency-aware footing.
\end{block}
