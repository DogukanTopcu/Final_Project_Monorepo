\chapter{CONCLUSIONS AND FUTURE WORK}

This term report documents a thesis project that is already substantial at the platform level, even though it is not yet complete at the experiment-validation level. The current implementation includes a local control plane, remote model deployment across multiple GPU tiers, a shared architecture contract, multiple architecture families, MLflow and report generation, a web-based experiment-management interface, and a measurement stack that records latency, tokens, cost, energy, and emissions. The project is also grounded in a completed systematic literature review that motivated the need for stronger operational measurement and for architecture-level comparison rather than model-only comparison.

At the same time, the report deliberately avoids over-claiming. The experiments are not yet valid for final thesis conclusions because the main coding benchmark is not yet finalized, the intended human-preference benchmark is not yet implemented in its final form, and the end-to-end runner path still requires cleanup before it can be considered a stable validation pipeline. The project-proposed swarm architectures are promising and relevant to the thesis novelty, but they must still be evaluated on a proper benchmark before any strong conclusion can be drawn about their comparative merit.

The next milestones are therefore clear. First, the \texttt{custom\_stratified} coding benchmark must be finalized and validated. Second, the experiment runner and its benchmark dependencies must be stabilized so that repeated comparative runs can be executed without environment friction. Third, the project should complete the first valid comparison cycle across monolithic, routing, multi-agent, and ensemble architectures, followed by the proposed swarm family. Finally, the human preference workflow should be integrated into the UI and used as a complementary evaluation layer. If these steps are completed, the project will be in a strong position to produce a final thesis chapter with defensible quantitative comparisons instead of exploratory platform evidence.
